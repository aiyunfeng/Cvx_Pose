Detailed Description

Pose estimation using the convex hull of SO(3) via a semidefinite program.

The semidefinite formulation allows us to include the point to plane metric as well as outlier rejection using a LASSO penalty. Currently, the publicly available SDPA package is used as our semidefinite solver.

Detailed Description

Iterative Closest Point.

Class that manages the ICP algorithm. It relies on a PCL class for performing correspondence, and a PoseEstimate instance for calculating the pose. ICP iterates betweeen these two functions to find a pose and correspondence between a model and an observation of that model.
